# CircleTone

音楽理論の 5 度表を用いた作曲アプリです。

## URL

申請中

## 使用技術

- TypeScript
- Node.js
- React Native
- Expo
- Firebase
  - Authentication
  - Firestore Database

## 機能一覧

- ユーザー登録、ログイン、ログアウト
- プロジェクト作成、削除機能
- Circle 機能（コード進行）
- Tone 機能（メロディー、ベース、ドラム）
